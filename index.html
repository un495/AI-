YouTube文字起こしプラン概要

使用ツール

  1. 会議の録画データ
  2. Googleアカウント
  3. Google Chrome ＋ YouTube Summary拡張機能
  4. Gemini

プロセス（3ステップ）

  1. YouTubeに動画を「限定公開」でアップロードする。
  2. YouTube Summary拡張機能で文字起こしデータを抽出・コピーする。
  3. Geminiに貼り付けて要約を生成する。

YouTube文字起こしプラン - 手順詳細①（動画アップロード）

YouTubeへ動画をアップロードし、セキュリティ設定を行います。

YouTubeアップロード手順

  1. YouTube Studioにログインする
  2. 「作成」→「動画をアップロード」をクリック
  3. 会議の録画ファイルをドラッグ&ドロップまたは選択
  4. 動画のタイトルを入力（例：「2024年1月定例会議」）
  5. 「詳細」タブで説明文を入力（任意）
  6. 「公開設定」タブに移動

【最重要セキュリティ設定】限定公開

動画のURLを知る関係者のみが視聴できる設定です。第三者による意図しない閲覧を防ぐため、必ずこの設定を行ってください。

設定手順：

  1. 「公開設定」タブで「限定公開」を選択
  2. 「保存」ボタンをクリック
  3. 動画のURLをコピーして保存（後で使用）

YouTube文字起こしプラン - 手順詳細②（文字起こし～要約）

拡張機能で文字起こしをコピーし、Geminiで要約します。

文字起こし手順（YouTube Summary拡張機能使用）

  1. アップロードした動画を開く
  2. YouTube Summary拡張機能のアイコンをクリック
  3. 「Transcript」タブを選択
  4. 表示された文字起こしテキストをすべてコピー

注意：YouTube Summary拡張機能は、YouTubeの字幕機能を利用して文字起こしを行います。動画に字幕が自動生成されている必要があります。

Gladia文字起こしプラン概要

【重要】Gladiaは英語サイトです

  3. Gemini アカウント

  1. Gladiaに音声・動画ファイルをアップロードする。
  2. 「話者分離」を有効にして、高精度な文字起こしを実行する。
  3. Geminiに貼り付け、高品質な要約を生成する。

Gladia文字起こしプラン - 手順詳細①（Gladiaの操作）

Gladiaで「話者分離」を有効にし、高精度な文字起こしを行います。

Gladia操作手順

  1. Gladiaにログインする
  2. 「New Transcription」をクリック
  3. 会議の音声・動画ファイルをアップロード
  4. 「Speaker diarization」のチェックボックスを有効にする
  5. 「Transcribe」ボタンをクリック
  6. 処理完了後、「Copy」ボタンでテキストをコピー

【ポイント】話者分離（Speaker diarization）

AIが発言者を自動で区別し、「Speaker 1」「Speaker 2」のように発言を整理する機能です。これにより、議事録の可読性と正確性が飛躍的に向上します。

Gladia文字起こしプラン - 手順詳細②（Gemini活用のコツと実践手順）

AIから理想的な出力を得るためのテクニックと、具体的な実践手順をご紹介します。

【コツ】Few-shot Prompting

AIに単に「要約して」と依頼するだけでなく、「望む出力形式の見本」を一緒に提示する手法です。

効果

AIが依頼者の意図を正確に理解し、期待に沿った形式で回答を生成。手戻りや修正の手間を大幅削減。

【実践手順】

  1. Gladiaで「Copy」ボタンをクリック
  2. Geminiを開く
  3. 下記テンプレートをコピー&ペースト
  4. 「# 会議の文字起こし」部分を置き換え
  5. 送信して結果を確認

推奨：会議ごとに新しいチャットを作成すると、履歴が整理しやすくなります。
